Para calcular a diminuição percentual, podemos utilizar a seguinte fórmula:


diminuição percentual = (valor inicial - valor final) / valor inicial x 100%


Substituindo pelos valores que temos, temos:


diminuição percentual = (60 - 22) / 60 x 100%

diminuição percentual = 38 / 60 x 100%

diminuição percentual = 0,6333 x 100%

diminuição percentual = 63,33%


Portanto, a diminuição percentual foi de 63,33%.
