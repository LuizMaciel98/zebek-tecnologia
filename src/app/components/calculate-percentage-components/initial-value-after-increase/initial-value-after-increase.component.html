Para calcular o valor inicial que aumentou em 10% e resultou em 220, podemos utilizar a seguinte fórmula:


valor inicial = valor final / (1 + percentual de aumento / 100%)


Substituindo pelos valores que temos, temos:


valor inicial = 220 / (1 + 10 / 100%)

valor inicial = 220 / 1,1

valor inicial = 200
