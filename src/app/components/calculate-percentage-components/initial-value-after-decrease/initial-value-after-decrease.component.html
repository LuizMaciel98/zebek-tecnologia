Para calcular o valor inicial que diminuiu em 18% e resultou em 200000, podemos utilizar a seguinte fórmula:


valor inicial = valor final / (1 - percentual de diminuição / 100%)


Substituindo pelos valores que temos, temos:


valor inicial = 200000 / (1 - 18 / 100%)

valor inicial = 200000 / 0,82
