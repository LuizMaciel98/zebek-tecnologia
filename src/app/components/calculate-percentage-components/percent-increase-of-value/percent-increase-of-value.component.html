Para calcular o aumento percentual, podemos utilizar a seguinte fórmula:


aumento percentual = (valor final - valor inicial) / valor inicial x 100%


Substituindo pelos valores que temos, temos:


aumento percentual = (1200 - 800) / 800 x 100%

aumento percentual = 400 / 800 x 100%

aumento percentual = 0,5 x 100%

aumento percentual = 50%


Portanto, o aumento percentual foi de 50%.
