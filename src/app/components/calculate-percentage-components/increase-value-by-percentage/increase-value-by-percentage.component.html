Para calcular o resultado do aumento de 28% em relação ao valor inicial de 120000, podemos utilizar a seguinte fórmula:


resultado = valor inicial + (percentual de aumento / 100%) x valor inicial


Substituindo pelos valores que temos, temos:


resultado = 120000 + (28 / 100%) x 120000

resultado = 120000 + 0,28 x 120000

resultado = 120000 + 33600

resultado = 153600
